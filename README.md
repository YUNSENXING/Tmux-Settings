# Tmux-Settings



The configuration files in this repository originate from **[AcWing](https://www.acwing.com)** and are shared here for personal use.

Download

1. Clone the repository 

```
git clone https://github.com/YUNSENXING/Tmux-Settings.git
```

2. Move files to home directory

```
cd /path/to/Tmux-Settings
mv Tmux-Settings/.bashrc Tmux-Settings/.tmux.conf Tmux-Settings/.vimrc ~/
```

3. Reload `.bashrc`

```
source ~/.bashrc
```



Usage

1. Once configured, you can start **tmux** directly

```
tmux
```

2. Split pane vertically

``` 
ctrl + "
```

![image-20251102192128610](D:\Algorithm\Tmux-Settings\assets\image-20251102192128610.png)

3. Split pane horizontally

```
ctrl + %
```

![image-20251102192223782](D:\Algorithm\Tmux-Settings\assets\image-20251102192223782.png)

4. Switch pane

```
(1) Mouse click
or
(2) Ctrl + A + arrow keys
```
